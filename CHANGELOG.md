# Changelog

Projedeki tüm önemli değişiklikler bu dosyada belgelenir.

## [1.0.0] - 2025-12-22

### Eklenenler
- İlk sürüm yayınlandı!
- 4 farklı oyun türü (Macera, Korku, Bilim Kurgu, Fantastik)
- Google Gemini AI entegrasyonu
- Dinamik hikaye üretimi
- Seçim bazlı gameplay
- Modern gradient UI
- Türkçe dil desteği
- Hikayeyi yeniden başlatma özelliği
- Ana menüye dönüş
- Loading göstergeleri
- Hata yönetimi
- Detaylı dokümantasyon

### Teknik Detaylar
- Flutter 3.0+ desteği
- Material Design 3
- State management yapısı
- Modüler kod organizasyonu
- Environment variables (.env)
- API güvenliği
