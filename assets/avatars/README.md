# Avatar Klasörü

Bu klasör gelecekteki avatar görselleri için ayrılmıştır.

## Mevcut Durum

Şu an oyun emoji tabanlı avatarlar kullanmaktadır. Hiçbir görsel dosya gerekmemektedir.

## Gelecek Geliştirmeler

İleride PNG/SVG formatında özel avatar görselleri eklenebilir:
- Karakter portreleri
- Tema bazlı avatarlar
- Özelleştirilebilir karakter tasarımları

## Not

Bu klasör şimdilik boş kalabilir. Emoji avatarlar kod içinde tanımlanmıştır.
